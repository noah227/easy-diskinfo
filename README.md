# easy-diskinfo

A Promise style disk information fetcher with many common used functions.

## Features

* Promise
* Declaration Support
* Flexible Functions
* Extended Functions

## Supported Platforms

- [ ] Windows
- [ ] Linux

## Usage

```js
const ndp = require("node-diskinfo-promise")

ndp.getDrives().then(drives => {
    
}).catch(e => {
    
})

```
